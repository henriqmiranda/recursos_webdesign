Fontes em CSS

  • font-family

    • Especifica a família de fontes para texto.

  • font-style

    • Especifica o estilo da fonte do texto.

  • font-size

    • Especifica o tamanho da fonte para o texto.

  • font-weight

    • Especifica se um texto deve ou não ser exibido em uma fonte small-caps.

  • font-variant

    • Especifica o peso de uma fonte.

Cores em CSS RGB

  • color: rgb(255, 255, 255);

    • Valores de cores RGB são suportados em todos os navegadores. Um valor de cor RGB é especificado com: rgb (vermelho, verde, azul). Cada parâmetro (vermelho, verde e azul) define a intensidade da cor como um inteiro entre 0 e 255. Por exemplo, rgb (0, 0, 255) é renderizado como azul, porque o parâmetro blue é definido como seu valor mais alto (255) e os outros são definidos como 0.

Cores em CSS RGBA

  • #p1 {background-color: rgba(255, 0, 0, 0.3);}

    • Valores de cor RGBA são uma extensão dos valores de cores RGB com um alpha canal - que especifica a opacidade de uma cor. Um valor de cor RGBA é especificado com: rgba (vermelho, verde, azul, alpha ). O alpha parâmetro é um número entre 0,0 (totalmente transparente) e 1,0 (totalmente opaco).

Posicionamento

  • posicion: absolute;

    • Este elemento serve para posicionamento de algo em relação e algo "mãe".

  • posicion: relative;

    • relative é o valor por defeito. Um elemento com position: relative; não é posicionado de uma forma qualquer. Diz-se que um elemento estático está não posicionado e um elemento com a sua posição com outro valor qualquer diz-se posicionado.

  • posicion: fixed;

    • Um elemento fixo é posicionado relativamente ao viewport, significa que fica sempre no mesmo sitio, mesmo que faça scroll da página.

  • posicion sticy

    • O posicionamento sticy é um híbrido de posicionamento relativo e fixo. O elemento é tratado como posicionamento relativo até cruzar um limite especificado, ponto no qual é tratado como posicionamento fixo.

  • float

    • O pretendido com float é envolver texto á volta de imagens.
